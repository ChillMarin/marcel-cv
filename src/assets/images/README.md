# Imagen de Perfil

Para agregar tu foto de perfil al CV:

1. **Nombra tu imagen como:** `marcel-profile.jpg`
2. **Colócala en esta carpeta:** `src/assets/images/`
3. **Formato recomendado:** JPG o PNG
4. **Tamaño recomendado:** Mínimo 200x200 píxeles (se redimensionará automáticamente)

La imagen se mostrará como un círculo perfecto de 100x100 píxeles con:

- Borde blanco elegante
- Sombra suave
- Efecto hover con zoom
- Recorte automático para mostrar solo tu cara

**Nota:** Si tu imagen tiene un nombre diferente, actualiza la propiedad `profileImage` en `cv.component.ts`
